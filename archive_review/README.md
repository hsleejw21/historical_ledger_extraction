# Archive Review Materials

This directory collects the paper-preparation materials developed around the archival corpus and data-construction workflow. It is intended to support documentation of source coverage, time conventions, page structure, and reviewer-facing validation questions.

## Contents

- `data_source/`
  - Draft text, tables, and freeze notes for a `Data Source` subsection centered on the Exeter College corpus.
- `audits/`
  - Coverage, page-count, year-range, and file-level time audits for the raw PDF corpus.
- `memos/`
  - Review memos, workflow sketches, and next-step checklists for preparing a journal-style data-preparation section.
- `figures/`
  - Annotated figure showing the typical structure of a ledger page.

## Scope

These materials are documentation artifacts rather than core pipeline code. They are included to make the archival side of the project easier to inspect, cite, and extend.

The raw archival PDFs are not duplicated here.
